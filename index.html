About Me.

Just a few things I've done and a few things I'm doing

In marketing science and analytics, I am most often interested in understanding how marketing efforts drive user behavior that leads to business impact. I've been in healthcare and tech my entire career, and usually bounce around the boundaries when looking for opportunities. I'm particularly interested in understanding signals and predictors that drive engagement which can be used to inform business decisions and better understand the user. I use a variety of analytical techniques like machine learning, computational models, and statistical analyses (most of which I'm classically trained in, but for the most part I like exploring and learning new techniques). In addition, my most recent projects have me building analytical infrastructures that improve technical capabilities and team workflow. I've been increasingly interested in building systems and frameworks that empower teams and individuals.

I am also an adjunct professor in Business Analytics and Information Systems in the School of Management at the University of San Francisco. I teach both undergraduates and MBAs analytical techniques to better facilitate data-driven decisions -- hopefully applying these techniques in industry.

Lastly, here are some essays on analytics and education. Updated semi-regularly.

    • Tools & Algorithms

    Import CSVs to Database Tool

  • A python tool that leverages Pandas to programmatically import CSV files to a Postgres database. This tool will convert all columns to lower case without spaces and autodetect datatypes Link to GitHub.

    • An API Manager to manage all your APIs, written in Flask

  • This app is an API that manages all your other APIs. The purpose is to easily and securely store all your API credentials (keys and curl path) so that users need not worry about all the different connections and keys needed to access different APIs. Link to GitHub.

    • Machine learning with Random Forest and Cross Validation

  • An introduction to machine learning using a random forest classifier and cross validation function. This project can also be repurposed using almost any machine learning algorithm in sklearn. Link to GitHub.

    • Logistic regression on email type to login

  • Performing a logistic regression to show the effectiveness of email types to user login, using python and statsmodels to run the regression. Link to GitHub.
